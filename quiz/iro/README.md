# 色彩図鑑MASTER（色名あて4択クイズ）

名画MASTER（`/quiz/meiga/`）と同じ構成の、色名を当てる4択ブラウザクイズです。
色見本を見て正しい色名を選びます。全5問・100点満点・Xシェア・リロードで別問題。

## 構成

- `index.html` … アプリ本体（単体で動作。ビルド不要）
- `colors.js` … 出題データ（自動生成）。`window.IRO_LEVELS` と `window.IRO_COLORS`

## データ

Notion「色彩図鑑」DB（i-iro.com カラー辞典より収集・計987色）を書き出したものです。

- 各色: `n`=名前 / `y`=よみ・綴り / `h`=webcolor / `f`=系統色（選択肢生成用）/ `d`=由来・概要 / `lv`=レベル
- レベルは `よみがな` の文字種から自動判定
  - Level 1「和の色」= 和色（ひらがな読み）267色
  - Level 2「世界の色」= イタリア・フランス・世界の色 720色
- レベルボタンは `colors.js` の `lv` 値から自動生成。**将来レベルを増やす場合はデータに `lv` を足すだけ**でボタンが増えます。

### データの更新

Notion側の色彩図鑑DBを更新したら、合図は **「クイズのデータを更新して」**。
`colors.js` を再生成します（名画MASTERと同じ運用）。

## 出題ロジック

- 各問、選択肢のダミー3件は **同じ系統色（`f`）** から優先的に選び、難易度を確保
  （同系統が足りない場合のみ全体から補完）
- 正解・ダミーは名前・色が重複しないように調整

## 画像素材（このフォルダに置くだけで反映）

`index.html` は以下のファイル名を参照します。`quiz/iro/` に置けば自動で反映されます
（`logo.png` が無い間はテキストロゴに自動フォールバック）。

| ファイル | 用途 | 推奨サイズ・形式 |
|---|---|---|
| `logo.png` | ヘッダーのロゴ（色タMASTER＋キュビコン） | 透過PNG・横640px前後 |
| `favicon.png` | ファビコン／ホーム画面アイコン（キュビコン顔） | 正方形・512×512・PNG |
| `og-card.png` | SNSシェア用サムネ（OGP） | 1200×630・PNG |

OGPサムネは「両サイドの色バー＋中央ロゴ＋『世界の色の名前クイズ』＋LEVEL★/★★/★★★」の構成。

## 広告（AdSense）

名画MASTERの知見に合わせた設置（クローラーはクイズを遊ばないため、
`hidden` な画面に置いた広告は認識されない）。

- 広告枠は**ページ最上部（ヘッダー直下・レベル選択の上）に常時表示**
- 名画MASTERと同じ `data-ad-client` / `data-ad-slot` を使用
- 読み込みは初期化時に**1回だけ** `adsbygoogle.push({})`
- 配信されない場合の次の一手：ページに短い読み物（色の豆知識など）を足して文字量を増やす

## デイリー（毎日コツコツの仕組み・第1弾）

名画MASTERの「デイリー化レシピ」を適用（Wordle方式・ログイン不要）。

- **今日の5色**：日付シード（`hashStr('iro-YYYY-MM-DD')`→`mulberry32`）で出題を決定。
  同じ日は全員同じ問題・翌日0時に自動で別問題。サーバー・毎日のアップロード不要。
  構成は段階式（和の色×2 → 世界の色×2 → 全色から1）。#N は `DAILY_EPOCH`(2026-07-24=#1) からの日数。
- **連続記録（ストリーク）**：`localStorage`（キー `iroMaster_state`。ドメイン共有のため接頭辞つき）。
  前回デイリーが昨日なら+1、それ以外は1にリセット。最終プレイが一昨日以前なら表示は0。
  **デイリーは1日1回のみ記録**。挑戦済みはカードから「きろく」へ誘導。
- **きろく（簡易マイページ）**：🔥連続／最高連続／プレイ回数／平均点＋直近にあそんだ色10件＋プレイ履歴。
- **結果シェア**：`🎨色名MASTER #N ○点 🔥n日連続` ＋URL（番号と連続日数が拡散の鍵）。
- **画面は4状態**：ホーム／出題／結果／きろく。初期表示はホーム（自動でクイズを始めない）。出題中は「← やめる」で中断可。
- **広告が埋まらないときは枠ごと自動で畳む**（空白の巨大な隙間を防止）。

> `colors.js` は他所（i-iro.com由来）データのため、データ自体を見せる図鑑コンプ・今日の色コラムは実装していません。
